# Online Courses System

This repository contains the documentation and screenshots of the Online Courses System developed as a running project for the Web Engineering course.

## Project Description
The Online Courses System is an e-learning platform that allows users to browse courses, register, sign in, enroll in courses, and take quizzes. The system also supports instructors and administrators to manage courses and system content.

## Project Features
- User Registration and Login
- Browse Online Courses
- Course Enrollment
- Instructor and Admin Management
- Secure Authentication

## Technologies Used
- WordPress (CMS)
- PHP
- MySQL
- HTML
- diagrams.net (for UML diagrams)

## Project Implementation
The system was implemented using WordPress as a Content Management System. Due to the nature of the CMS-based implementation, the core system logic is handled by WordPress plugins and themes.
